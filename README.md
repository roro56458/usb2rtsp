# Welcome / Bonjour
Bienvenue sur le projet **usb2rtsp**.  
Ce projet permet de transformer votre caméra USB en un flux **RTSP** compatible avec **Home Assistant, Frigate**, ou tout autre serveur NVR compatible RTSP.
